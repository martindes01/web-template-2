• web-template-2

    The header and footer of this page are loaded dynamically using AJAX. Find out more here.

  • How it works

    This page uses an HTML data attribute to specify the paths to header.txt and footer.txt. These files contain the HTML markup for the header and footer of this page. 

    <body data-ajax-text-files="header.txt footer.txt"></body>

    Placeholder elements are given data attributes to specify the inclusion points for these files. 

    <body data-ajax-text-files="header.txt footer.txt">
    <table data-ajax-text-file="header.txt"></table>
    ...
    <table data-ajax-text-file="footer.txt"></table>
</body>

    An asynchronous JavaScript (AJAX) function is called to load these files dynamically, append their contents to the inner HTML of their respective placeholder elements and execute the specified callback function. 

    <body data-ajax-text-files="header.txt footer.txt"
      onload="Ajax_Document_StartText(function () {
          /* callback function */
      });">
    <table data-ajax-text-file="header.txt"></table>
    ...
    <table data-ajax-text-file="footer.txt"></table>
</body>

    This framework can also be used to retrieve tags from XML files.
